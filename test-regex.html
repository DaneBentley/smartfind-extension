SmartFind Regex Test Page

How to Test Regex Functionality

Use the SmartFind extension with the * prefix to test regex patterns:

  • *test - Find all occurrences of "test"
  • *\d+ - Find all numbers
  • *[A-Z][a-z]+ - Find capitalized words
  • *\b\w{4}\b - Find 4-letter words
  • *email|phone|address - Find any of these words
