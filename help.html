Everything you need to know about using SmartFind

Getting Started

Installation

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome" to install SmartFind
  3. The extension will be ready to use immediately
  4. Press Ctrl+F (or Cmd+F on Mac) to activate

First Search

Try these examples to get started:

/ Where is the contact info?
AI search for natural questions
contact, email, phone
Multi-term search with colors

Search Modes

AI Search (/)

Ask questions in natural language. Perfect for finding content by meaning rather than exact words.

Examples:

/ Where is the pricing information?
/ How do I contact support?
/ What are the system requirements?
/ Find the author bio
Cost: 1 token per AI search • 50 free tokens per month

Keyword Search (default)

Traditional exact keyword matching. Fast and precise for known terms.

Examples:

contact
JavaScript
API documentation
Cost: Always free and unlimited

Multi-term Search (,)

Search for multiple terms simultaneously with color-coded highlights.

Examples:

price, cost, fee
contact, email, phone, support
JavaScript, Python, React
Cost: Always free and unlimited

Regex Search (*)

Use regular expressions for powerful pattern matching.

Examples:

*\d{3}-\d{3}-\d{4} // Phone numbers
*[a-z]+@[a-z]+\.[a-z]+ // Email addresses
*https?://\S+ // URLs
*\$\d+(\.\d{2})? // Prices
Cost: Always free and unlimited

Troubleshooting

Common Issues

SmartFind doesn't activate

  • • Make sure the extension is installed and enabled
  • • Try refreshing the page
  • • Check if the page allows extensions (some internal pages don't)
  • • Try clicking the extension icon in the toolbar

AI search not working

  • • Check if you have tokens remaining
  • • Make sure you're signed in for AI features
  • • Verify your query starts with /
  • • Try a simpler question first

Search results not highlighting

  • • Some websites may block styling modifications
  • • Try different search terms
  • • Check if content is inside iframes
  • • Refresh the page and try again
